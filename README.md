# E-Commerce Sales Dashboard Analysis

## Project Overview

This project analyzes e-commerce sales data using Power BI to track revenue, profit, and business performance metrics. The dashboard provides interactive visualizations to support data-driven decision-making.

## Tools Used

* Power BI
* CSV Dataset
* Data Visualization
* Business Intelligence

## Dashboard Features

* Revenue and Profit Analysis
* Category-wise Sales Performance
* Regional Sales Analysis
* Top-Selling Products
* KPI Tracking
* Interactive Filters and Visualizations

## Analysis Performed

* Data Cleaning and Transformation
* Data Modeling
* KPI Development
* Trend Analysis
* Dashboard Design and Visualization

## Key Insights

* Identified top-performing product categories and products.
* Analyzed regional sales and profit trends.
* Tracked revenue performance using interactive KPIs.
* Developed business insights through visual reporting.

* Power BI Dashboard (.pbix)
